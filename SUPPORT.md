# Support

## GitHub Discussions (preferred)

For questions, ideas, and general conversation use **[GitHub Discussions](https://github.com/jomkz/fighters-legacy/discussions)**.

| Category | Use for |
|---|---|
| Announcements | Release and project news (maintainer only) |
| General | Anything that doesn't fit elsewhere |
| Q&A | Questions about building, running, or modding |
| Ideas | Feature brainstorming before filing a formal request |
| Show & Tell | Community screenshots, mods, experiments |

## GitHub Issues

Use issues for **bug reports** and **feature requests** only, via the provided templates. Do not ask support questions in issue comments.

- [Report a bug](.github/ISSUE_TEMPLATE/bug_report.yml)
- [Request a feature](.github/ISSUE_TEMPLATE/feature_request.yml)
- [Propose an RFC](.github/ISSUE_TEMPLATE/rfc.yml)

## Response Time

This is a volunteer-driven project. There is no SLA. Best-effort responses are the norm.

## Escalation

For Code of Conduct concerns or sensitive matters, email **fighters-legacy@mkz.io**.
